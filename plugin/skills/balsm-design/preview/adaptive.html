§6.5 · One scale, all stacks

Responsive & adaptive

Balsm spans phone → tablet → desktop POS → web admin → widgets → print. The same numeric scale maps to Flutter LayoutBuilder, Tailwind sm/md/lg/xl, and CSS @media / @container.

Responsive

Fluid reflow across a continuous range — driven by the viewport. Marketing web, admin UI. Use .container, .grid, .card-grid, fluid type.

Adaptive

Distinct layouts swapped at a boundary — driven by the element's own container, so it adapts inside a narrow pane / foldable. Patient app, POS. Use .cq + .adaptive-*.
